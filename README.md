## Para Rodar o Projeto

1. **Configure o .env**

2. **Instale as dependências do PHP:**

   ```bash
   composer install
   ```

3. **Gere a chave da aplicação:**

   ```bash
   php artisan key:generate
   ```

4. **Execute as migrations:**

   ```bash
   php artisan migrate
   ```

5. **Inicie o servidor Laravel:**

   ```bash
   php artisan serve
   ```
   
6. **Acesse o projeto no navegador:**

   ```
   http://127.0.0.1:8000
   ```
